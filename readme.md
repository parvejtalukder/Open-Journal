# 💰 CS50 Finance

A stock trading web app built with Flask for Harvard's CS50x.

## ✨ Features
- 🔐 User authentication
- 📈 Real-time stock lookup
- 💵 Buy/sell stocks with virtual cash
- 📊 Portfolio overview
- 🧾 Transaction history

## 🛠️ Tech Stack
- **Backend:** Python (Flask)
- **Database:** SQLite
- **Frontend:** HTML, CSS, Bootstrap

## 📦 Quick Start
```bash
git clone https://github.com/parvejtalukder/CS50-finance-Flask-.git
cd CS50-finance-Flask-
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
flask run
```

## 📁 Structure
- `app.py` - Main application
- `helpers.py` - Utility functions
- `static/` - CSS files
- `templates/` - HTML pages

## 📝 Notes
- Users start with $10,000 virtual cash
- `finance.db` generated locally on first run
- Educational project for CS50
